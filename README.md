# SEO Page Analyzer

Web-service for check status of websites. [Live Demo](https://seo-page-analyzer.kaziamov.com/)


## Features

* Check status code
* Parse text from title tag
* Parse text from h1
* Parse meta-description


## Install

```powershell
make install
```

## Usage

```shell
make start
```
